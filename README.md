# ❤️ NeoVitalPulse – Heart Disease Prediction System

NeoVitalPulse is a Machine Learning based healthcare prediction system that analyzes patient medical data to predict the risk of heart disease.

Heart disease remains one of the leading causes of death worldwide. Early detection can significantly reduce mortality rates. This project uses machine learning algorithms to analyze medical attributes and predict whether a person is likely to develop heart disease.

The goal of this project is to demonstrate how Artificial Intelligence can assist healthcare professionals in early diagnosis and preventive care.

---

## 🚀 Project Overview

NeoVitalPulse predicts heart disease using various clinical parameters collected from patients. The system processes health-related attributes and generates a prediction indicating the likelihood of heart disease.

The prediction is based on machine learning models trained on medical datasets containing multiple cardiovascular indicators.

## 📊 Dataset Description

The dataset contains various medical attributes of patients used to predict heart disease.

Key Features Include:

- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol Level
- Fasting Blood Sugar
- Resting ECG Results
- Maximum Heart Rate Achieved
- Exercise Induced Angina
- ST Depression
- Slope of Peak Exercise ST Segment
- Thalassemia
- Target (Heart Disease / No Heart Disease)

These attributes are used to train machine learning models for accurate predictions.

## 🧠 Machine Learning Workflow

The project follows a structured machine learning pipeline.

### 1. Data Collection
The dataset contains patient medical records related to cardiovascular health.

### 2. Data Preprocessing
- Handling missing values
- Data cleaning
- Encoding categorical variables
- Feature scaling and normalization

### 3. Exploratory Data Analysis (EDA)
To understand patterns in the dataset, the following analyses were performed:

- Statistical summaries
- Feature distribution analysis
- Correlation heatmap
- Visualization of medical attributes

### 4. Feature Engineering
Important health indicators were selected to improve prediction performance.

### 5. Model Training
Multiple machine learning algorithms were tested including:

- Logistic Regression
- Decision Tree
- Random Forest
- Support Vector Machine

### 6. Model Evaluation
Model performance was evaluated using:

- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC Score

---

## 📈 Model Performance

The trained model achieved strong prediction performance.

**Accuracy:** 87.31%

**ROC-AUC Score:** 0.95

### Classification Report

| Class | Precision | Recall | F1 Score |
|------|-----------|--------|---------|
| No Heart Disease | 0.92 | 0.81 | 0.86 |
| Heart Disease | 0.83 | 0.93 | 0.88 |

A high ROC-AUC score indicates that the model performs well in distinguishing between patients with and without heart disease.
